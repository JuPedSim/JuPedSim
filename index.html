JuPedSim

Simulating pedestrians with state of the art models. The JÃ¼lich Pedestrian Simulator (JuPedSim) consists of three modules which are loosely coupled and can be used independently at the moment. These are:

  1. JPScore: the core module computing the trajectories. site of project.
  2. JPSvis: a tool for visualizing the input (geometry) and output (trajectories) data (not maintained).
  3. JPSreport: a tool for analyzing the trajectories. site of project.
  4. JPSeditor: a tool for creating and editing geometry files.

The code is actively developed in a GitLab repository. Please consider cloning the code there.
